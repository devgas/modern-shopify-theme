# Modern Shopify Theme

A clean Online Store 2.0 Shopify theme scaffold with responsive Liquid templates, configurable sections, product cards, collection grids, cart drawer, customer account pages, and theme settings.

## Local Checks

```bash
shopify theme check
```

## Development

```bash
shopify theme dev --store your-store.myshopify.com
```

The theme uses standard Shopify folders: `layout`, `templates`, `sections`, `snippets`, `assets`, `config`, and `locales`.
